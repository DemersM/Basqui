Basqui 1.0 - Cloud Based Web Mapping GIS


Current Functionnalities:


  • Import/Export of multiple GIS vector file format such:
    • ESRI Shapefile
    • MapInfo TAB
    • GeoJson
    • KML/KMZ
    • SQLite/Spatialite
    • CSV
  • Creation and editing vector features:
    • Use of OpenLayers vector tools
  • Connect to Web Sercices such:
    • WMS
  • Handful table attribute speadsheet allowing to:
    • Filter and order data
    • Edit data
    • Manage table schema
    • Data validation
  • Create tiled maps with:
    • Customization of layers styling
    • Customization of labeling
    • Customization of map options
  • Helpful real-time application log system
    • Simple and efficient project classifying
